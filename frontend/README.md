Frontend (React + Vite + Tailwind)

This frontend is a demo RCB-themed UI built with React, Vite and TailwindCSS.

Setup:

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173

Notes:
- The UI contains placeholder images and sample data.
- To connect with the backend, update API calls in the components/pages to point to `/api` or your backend URL.
